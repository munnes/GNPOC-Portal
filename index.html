Corporate Intranet Portal

Project Definition:

An enterprise portal, which is a Web-based interface for users of enterprise applications. It also provide access to enterprise information such as corporate databases, applications (including Web applications), and systems

Portal was developed using Asp.net and SQL database, also it has integration between oracle e business suite r12.1. (ERP) and third party systems.

Corporate solutions:

Portal has a list of corporate solutions built as per customer request and needs, I'll mention below some of those systems

  • Security pass system: This system provide a security pass for contractors, pass is issued after completing the approval cycle
  • Management meeting schedule (MMS): In this system top management meeting are scheduled meeting papers are uploaded and MOMs can be added
  • Field base management system (FBMs): This system serves field people in many activities such as accommodation
  • Aviation system: : This system takes care of internal travel
